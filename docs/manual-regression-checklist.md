# 手工回归清单

## 1. 启动

- [ ] `npm start` 可启动
- [ ] 页面可访问
- [ ] 健康检查可访问

## 2. 鉴权

- [ ] 输入 token 后可连接
- [ ] token 错误时出现鉴权失败提示

## 3. 会话

- [ ] 新建会话成功
- [ ] 选择会话后可同步历史
- [ ] 关闭窗口后会话仍在列表中
- [ ] 刷新页面后会话仍可恢复

## 4. 消息

- [ ] 发送纯文本消息成功
- [ ] 流式输出可见
- [ ] turn 结束状态正确

## 5. 审批

- [ ] 审批卡片出现
- [ ] 可以提交审批响应
- [ ] session 级批准行为可用

## 6. request_user_input

- [ ] 表单能正确渲染
- [ ] 提交后服务端能接收

## 7. 上传

- [ ] 图片可上传
- [ ] 上传后可作为附件发送
- [ ] 图片预览正常

## 8. 工作区

- [ ] 可浏览目录
- [ ] 可切换目录
- [ ] 可新建目录
- [ ] 可使用当前目录创建会话

## 9. 连接

- [ ] 断线后能自动重连
- [ ] 重连后 active thread 可自动 sync

## 10. 本地窗口
- [ ] 可确认不再依赖旧本地窗口恢复兼容逻辑
