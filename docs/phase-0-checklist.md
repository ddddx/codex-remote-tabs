# 阶段 0 清单

## 1. 目标

在真正开始重构前，完成现状盘点、行为冻结与回归基线记录。

## 2. 输入

- 当前仓库
- 当前运行脚本
- 当前前端页面
- 当前后端 API / WS 实现

## 3. 输出

必须产出以下文件：

- `docs/current-behavior.md`
- `docs/api-inventory.md`
- `docs/ws-inventory.md`
- `docs/manual-regression-checklist.md`
